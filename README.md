隐私政策

感谢您使用本应用。本应用注重用户的隐私保护，制定本隐私政策以说明我们如何收集、使用、存储及保护您的个人信息。

1. 信息收集

我们可能收集以下信息：

• 设备信息、网络状态信息

• 用户主动提供的昵称、头像等账号信息

• 使用过程中产生的对话内容、使用记录

• 苹果内购订阅相关的交易记录（仅用于功能验证）

2. 信息使用

收集的信息仅用于：

• 提供AI对话服务

• 账号登录、数据同步

• 订阅状态验证、功能解锁

• 优化服务体验、故障排查

3. 信息共享

我们不会向第三方共享、出售或转让您的个人信息，
除非法律要求或为完成订阅支付所必需。

4. 用户权利

您可以：

• 查看、更正个人信息

• 注销账号

• 取消订阅（在苹果账户设置中操作）

5. 订阅与自动续费

本应用包含自动续订订阅：

• 订阅周期：按月/按年

• 订阅将自动续费，除非在当前周期结束前24小时关闭

• 用户可在账户设置中管理或关闭自动续订

6. 信息安全

我们采取合理的技术措施保护用户信息安全，
防止泄露、丢失或滥用。

7. 政策更新

我们可能不定期更新隐私政策，更新后将在应用内公示。

8. 联系我们

如对隐私政策有疑问，可通过应用内客服渠道联系。
